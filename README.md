# QuickKeyLauncher
QuickKeyLauncher is a lightweight Windows utility that lets you launch files, folders, and URLs instantly using customizable hotkeys.  Main features: - Register any file, folder, or URL with a hotkey - Fast and minimal UI - No background processes when idle  Download the installer (QuickKeyLauncher_Setup_V1.1.exe) and run it to install.
